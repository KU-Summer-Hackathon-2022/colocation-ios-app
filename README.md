### 개발환경

앱 프로젝트의 개발환경은 다음과 같습니다.

 - UIKit 기반
 - iOS 15.5 이상
 - XCode 13.4.1
 - Swift 5.6

### 사용하는 스위프트 패키지

앱 프로젝트에 사용되는 스위프트 패키지는 다음과 같습니다.
 
 - RxSwift 6.5.0 이상
 - SnapKit 5.6.0 이상


### 기능
 - 지도의 Annotation을 통해 장소를 바로 알 수 있어요.
 - 지도의 핀을 클릭하면 상세 페이지를 확인할 수 있어요.
 - 상세 페이지에서 해당 숙소를 신청할 수 있어요.
 - 내가 원하던 정보를 북마크로 모아서 관리할 수 있어요.
